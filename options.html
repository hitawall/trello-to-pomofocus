Trello to Pomofocus

Settings

1 Trello API Key & Token

Get your API key from trello.com/app-key

Trello opened in a new tab. After you click Allow, Trello will show your token on screen. Copy it and paste it below.

2 Select Board

Choose the Trello board to sync tasks from.

3 List Names to Sync

Comma-separated list names to include. Cards in these lists will be synced.

Matching is case-insensitive. Example: To Do, In Progress, Doing
